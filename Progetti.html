WebCloud

E' stato il mio primo progetto scolastico in cui ho partecipato come Web Developer, era un template fatto semplicemente in HTML & CSS dove c'era una semplicissima interfaccia web con il sito e un paio di animazioni. Il sito web non l'ho completamente fatto io e sono state apportate varie modifiche da un altro sviluppatore web.

WebCloud Library

WebCloud Library è il seguito del sitoweb WebCloud, il mio ruolo in questo progetto era sia il Project Manager che il main developer, grazie all'aiuto di altre persone siamo riusciti a creare un design minimale e perfetto per una libreria scolastica. Il sito è stato sviluppato con i seguenti linguaggi: HTML, CSS, JavaScript. Ovviamente era un pò più completo del precedente soltanto che rimane sempre un semplicissimo template statico.
